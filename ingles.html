LINEA DEL TIEMPO EN INGLES

1. *Década de 1990:* Se lanzan los primeros navegadores web, como Mosaic y Netscape. Las páginas web son principalmente texto y enlaces hipertexto.

2. *1993:* Se crea el primer motor de búsqueda, W3Catalog, lo que facilita la búsqueda de páginas web.

3. *1994:* Aparece el lenguaje de marcado HTML 2.0, que introduce características como formularios y tablas.

4. *1995:* Netscape lanza JavaScript, permitiendo la creación de páginas web interactivas. Además, se lanza Internet Explorer.

5. *1996:* CSS (Cascading Style Sheets) se introduce para separar el diseño de la estructura de una página web.

6. *1998:* Se lanza Google, revolucionando la búsqueda en línea. También se publica HTML 4.0.

7. *2000:* Se inicia la burbuja de las puntocom y el auge de las empresas de tecnología.

8. *2004:* Se lanza Mozilla Firefox, un navegador alternativo a Internet Explorer.

9. *2005:* YouTube se funda, marcando el inicio de la era del video en línea.

10. *2007:* Apple lanza el iPhone, cambiando la forma en que se accede a las páginas web con un enfoque en la navegación móvil.

11. *2010:* HTML5 se convierte en un estándar web, introduciendo nuevas capacidades multimedia y mejoras en la accesibilidad.

12. *2015:* La tecnología de páginas web progresivas (PWA) gana popularidad, brindando experiencias web más rápidas y eficientes.

13. *2020:* La pandemia de COVID-19 impulsa la importancia de las páginas web para la comunicación y el comercio en línea.
